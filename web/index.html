This app compares the performance of creating a Base64 string for a 1MB of data (a JavaScript Uint8Array of 1,000,000 bytes).

The image on the left is based on the first 1200 and last 1200 bytes from the source byte array (using 4x8-byte RGBA values).

These bytes are sent to the custom JavaScript function that creates a Base64 string from the bytes. This Base64 string is then converted back to bytes using the built-in atob() function, and the center image is constructed using those bytes.

Then the bytes are sent to a WebAssembly Rust function that creates a Base64 string from the bytes. This Base64 string is then converted back to bytes using the built-in atob() function, and the right-hand image is constructed using those bytes.
